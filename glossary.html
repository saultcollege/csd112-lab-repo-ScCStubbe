Glossary

HTML
Hypertext Marked Language.
CSS
Cascading Style Sheets
Favicon
A favicon is the emblem or logo that is shown on the top tab to make the website more recognizable to users whith multiple tabs open
<hr> element
Creates a horizantal seperating line through the page
<br> element
Used to cause a line break
<pre> element
Used to show preformaated text, causes the browser to render every whitespace.
<!-- , --!>
The indicators for commenting in HTML code, <!-- for opening the comment space and !--> for closing
